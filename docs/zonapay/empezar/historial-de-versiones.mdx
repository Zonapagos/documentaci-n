<Update label="v6.1 — Abril 2026" description="Documentación Mintlify (esta publicación)">

### Cambios en la documentación

- Nueva estructura de navegación en Mintlify con tabs separadas: Empezar, Guías, API Reference, Recetas, Soporte.
- Glosario de siglas agregado al inicio del documento.
- Guía rápida de 10 minutos con código en cURL, Node.js, Python.
- Diagramas de secuencia con Mermaid para el flujo completo y la sonda.
- Parser de referencia completo para `str_res_pago` en JavaScript, Python y C#.
- FAQ con los 15 errores más comunes al integrar.
- Tabla consolidada de todos los estados de `int_estado_pago` con acciones recomendadas.
- Endpoints separados para `InicioPago` y `VerificacionPago` con ejemplos en Postman embebidos.

### Hallazgos pendientes de validación con TI

- Valor correcto de `int_modalidad` (doc oficial tiene contradicción: texto dice `-1`, ejemplo JSON usa `1`).
- Disponibilidad de un endpoint REST sandbox público.
- Normalización de mensajes de error (actualmente se expone `"Object reference not set…"` crudo).
- Documentación del algoritmo de hash del token `rut` en la URL del ciclo de pago.

</Update>
