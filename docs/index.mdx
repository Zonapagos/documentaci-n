# 👋 Bienvenido a Zonapagos

Bienvenido al portal para desarrolladores de Zonapagos.

Este portal está diseñado para equipos de desarrollo que necesitan integrar los servicios de pago y recaudo mediante API.

---

## 🚀 ¿Qué es Zonapagos?

Zonapagos es una plataforma tecnológica para habilitar, procesar y administrar pagos digitales.

Permite conectar comercios, plataformas y aliados a diferentes servicios de recaudo y pago mediante integraciones seguras, escalables y trazables.

---

## ⚙️ Modelo de negocio

### Gateway
Para clientes que requieren integración técnica y control del flujo transaccional.

### Agregador
Para operaciones centralizadas que buscan acelerar la habilitación.

---

## 🧩 Modalidades de integración

### 🟢 Zonaexpress
Formulario web alojado por Zonapagos para recaudo sin desarrollo.

**Características:**
- Experiencia alojada
- Implementación rápida
- Sin frontend propio

---

### 🟣 Zonarecaudo
Gestión completa del ciclo de recaudo.

**Opciones:**
1. Portal alojado (sin integración)
2. API (consulta en tiempo real)

---

### 🔵 Zonapay
Pasarela de pagos para web o móvil.

**Características:**
- Checkout optimizado
- Integración rápida
- Soporte múltiples medios

---

## 🔗 Complementos

### Link de cobro
Generación de enlaces de pago.

### QR de pago
Cobros mediante códigos QR.

### Tokenización
Protección de datos sensibles usando tokens.

### Dispersión de fondos
Distribución automática de dinero entre cuentas.

---

## 🧠 Flujo de implementación

1. Entendimiento del servicio  
2. Integración técnica  
3. Validación  
4. Salida a producción
